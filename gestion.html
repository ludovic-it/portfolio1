Site de Gestion du Personnel

Application web de gestion du personnel développée en PHP avec architecture MVC

Démonstration

Présentation du projet

Dans le cadre de ma formation en BTS SIO option SLAM, j'ai développé un site web de gestion du personnel en PHP avec une architecture MVC (Modèle-Vue-Contrôleur). L'application utilise une base de données MySQL via phpMyAdmin et Bootstrap pour assurer le responsive de l'interface utilisateur.

Les mots de passe sont hashés avec SHA-256 avec salage pour sécuriser le stockage des mots de passe utilisateurs.

Architecture MVC du projet

1. Page d'accueil (index.php)

La page d'accueil sert de point d'entrée à l'application et redirige automatiquement l'utilisateur vers la page de connexion. 

case "login":
default:
    require_once "controleurs/C_login.php";
    $controleur = new C_login();
    $controleur->action_afficher();
    break;

Page d'accueil - Redirection vers la connexion

2. Système d'authentification

2.1 Page de connexion (login.php)

L'utilisateur arrive sur cette page et dispose de deux options : se connecter avec des identifiants existants ou s'inscrire pour créer un nouveau compte.

Page de connexion avec options de connexion et d'inscription

2.2 Inscription

Lors de l'inscription, le système collecte les informations utilisateur, hache le mot de passe avec un salage SHA-256 et stocke les données sécurisées en base de données.

Formulaire d'inscription

2.3 Page d'accueil utilisateur

Après connexion réussie, l'utilisateur accède à la page d'accueil qui possède plusieurs fonctionnalités : un accès via un bouton déroulant "Liste des employés" et l'affichage des employés stockés en base de données sous forme de tableau.

Page d'accueil utilisateur avec tableau des employés

3. Fonctionnalités principales

3.1 Ajout d'employés

Fonctionnalité permettant d'ajouter de nouveaux employés via un formulaire de saisie des informations employé.

Formulaire de saisie des informations employé

3.2 Déconnexion

Un bouton de déconnexion est accessible depuis l'interface principale, permettant une gestion sécurisée de la session utilisateur.

4. Interface utilisateur

4.1 Site responsive

Le site utilise Bootstrap pour une navbar responsive ainsi que des boutons et des tableaux responsives pour l'affichage des données.

4.2 Éléments d'interface

L'interface comprend une barre de navigation principale avec menu responsive, des boutons au design cohérent avec Bootstrap, et des tableaux pour une présentation claire des données employés.

Interface responsive avec Bootstrap

5. Base de données et sécurité

5.1 Sécurité des données

Les mots de passe sont hachés avec SHA-256 et salage pour garantir la sécurité des données utilisateurs. 

$mdp_hashed = password_hash($mdp, PASSWORD_DEFAULT);

// Vérification du mot de passe
if (password_verify($mdp, $user['mdp']))
{
    $_SESSION['login'] = $login_safe;
    header("Location: index.php?page=accueil");
    exit();
}
else
{
    $this->action_afficher("Mot de passe incorrect.");
}

password_verify permet de vérifier si les mots de passe rentrés en paramètre sont valides et affiche donc la page d'accueil en cas de succès.

7. Conclusion

Ce site de gestion du personnel offre une architecture MVC bien structurée, une sécurité pour l'authentification avec hashage SHA-256, une interface responsive grâce à Bootstrap, des tableaux pour la gestion des employés et un formulaire pour ajouter un nouvel employé.

Ce projet m'a permis de renforcer mes compétences en développement web avec PHP, en architecture MVC, en gestion de bases de données MySQL et en sécurité web.
