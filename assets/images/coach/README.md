# Images pour l'application Coach

Ce dossier contient les images nécessaires pour l'application mobile Coach.

## Images requises

1. **coach-hero.png** (1200x600px)
   - Image principale de l'application mobile Coach
   - Représente une application mobile de fitness/coaching
   - Format : PNG ou JPG

2. **coach-interface.png** (800x500px)
   - Capture d'écran de l'interface utilisateur
   - Montre les champs de saisie et les résultats
   - Format : PNG ou JPG

## Suggestions d'images générées par IA

Les images peuvent être générées avec des outils d'IA comme :
- DALL-E
- Midjourney
- Stable Diffusion
- Bing Image Creator

### Prompts suggérés

1. Pour coach-hero.png :
   - "Modern mobile fitness app interface on smartphone screen, clean design, health and fitness theme, gradient background, professional UI/UX design"

2. Pour coach-interface.png :
   - "Mobile app screen showing BMI/body fat calculator interface, input fields for weight height age, gender selection buttons, result display with smiley emoji, modern flat design"
