Coach

Application mobile multiplateforme développée avec .NET MAUI permettant de calculer l'Indice de Masse Grasse (IMG)

Présentation du projet

Dans le cadre de ma formation en BTS SIO option SLAM, j'ai développé une application mobile multiplateforme Coach permettant de calculer l'Indice de Masse Grasse (IMG) d'un utilisateur en fonction de son poids, sa taille, son âge et son sexe. L'objectif était de créer une application de coaching fitness et santé accessible sur toutes les plateformes mobiles et desktop.

L'application utilise .NET MAUI (Multi-platform App UI) version 10.0 pour garantir une expérience utilisateur cohérente sur Android, iOS, macOS et Windows. Elle permet aux utilisateurs de suivre leur composition corporelle et d'obtenir des recommandations personnalisées basées sur leur profil.

Fonctionnalités principales

L'application permet aux utilisateurs de saisir leurs informations personnelles (poids, taille, âge, sexe) via une interface intuitive et de calculer instantanément leur Indice de Masse Grasse. Le résultat est affiché avec un message d'interprétation personnalisé et un indicateur visuel (smiley) pour une compréhension rapide.

L'interface utilisateur est conçue pour être simple et efficace, avec des champs de saisie numériques optimisés pour mobile, une sélection intuitive du sexe via des RadioButtons, et un affichage clair des résultats avec des messages explicatifs.

Architecture

L'application utilise une architecture MVVM partielle avec .NET MAUI pour séparer la logique métier de l'interface utilisateur. Le traitement des données et les calculs sont effectués dans le modèle Profil, tandis que l'interface est définie en XAML pour une séparation claire des responsabilités.

Point d'entrée de l'application

Le fichier MauiProgram.cs initialise l'application MAUI en configurant les services nécessaires, en enregistrant les polices personnalisées (OpenSans-Regular, OpenSans-Semibold) et en configurant le logging en mode DEBUG. La classe App.xaml.cs gère le cycle de vie de l'application et crée la fenêtre principale avec AppShell.

Développement

Interface utilisateur (MainPage.xaml)

La page principale contient tous les éléments nécessaires pour le calcul de l'IMG :

  • Champs de saisie : Poids (kg), Taille (cm), Âge (années) avec clavier numérique
  • Sélection du sexe : RadioButtons pour homme/femme (femme sélectionné par défaut)
  • Bouton de calcul : Lance le calcul de l'IMG
  • Affichage des résultats : Image smiley et label avec le message d'interprétation

Interface utilisateur avec champs de saisie et résultats

Modèle de données (Profil.cs)

La classe Profil encapsule la logique de calcul de l'IMG. Elle contient :

  • Les propriétés privées pour stocker les données utilisateur (sexe, poids, taille, âge)
  • La méthode CalculIMG() qui implémente la formule de Deurenberg
  • La méthode ResultatIMG() qui détermine le message d'interprétation
// Formule de Deurenberg
// IMG = (1.2 × poids / taille²) + (0.23 × âge) - (10.83 × sexe) - 5.4
// où sexe = 1.0 pour homme, 0.0 pour femme
// taille convertie en mètres (taille / 100.0)

public void CalculIMG()
{
    double tailleEnMetres = taille / 100.0;
    double sexeValue = sexe ? 1.0 : 0.0;
    img = (1.2 * poids / (tailleEnMetres * tailleEnMetres)) 
          + (0.23 * age) 
          - (10.83 * sexeValue) 
          - 5.4;
}

Interprétation des résultats

Les seuils d'interprétation varient selon le sexe :

Femmes

  • IMG < 25% : "maigre"
  • 25% ≤ IMG ≤ 30% : "Parfait"
  • IMG > 30% : "Surpoids"

Hommes

  • IMG < 15% : "Trop maigre."
  • 15% ≤ IMG ≤ 20% : "Parfait."
  • IMG > 20% : "Surpoids."

Gestion des erreurs

L'application inclut une gestion des erreurs avec try-catch pour capturer les erreurs de parsing des données saisies. En cas de saisie incorrecte, une alerte est affichée à l'utilisateur pour l'informer du problème.

Notes techniques

Formule IMG

La formule utilisée est celle de Deurenberg (1991) pour le calcul de l'Indice de Masse Grasse (IMG) : 

IMG = (1.2 × IMC) + (0.23 × âge) - (10.8 × sexe) - 5.4

où :

  • IMC = poids (kg) / taille² (m)
  • sexe = 1 pour homme, 0 pour femme

Interprétation des résultats

Les seuils d'interprétation varient selon le sexe et sont basés sur les recommandations médicales standard :

Femmes

Normal entre 25% et 30%

Hommes

Normal entre 15% et 20%
