# Documentation des Projets - Portfolio Ludovic.C

## 1. SIO-SHOES - Site E-commerce Symfony

**Technologies :** Symfony, PHP, MySQL, Stripe, Twig

Site e-commerce complet de vente de chaussures développé dans le cadre du BTS SIO option SLAM. Le site permet aux utilisateurs de naviguer parmi les produits classés par catégories, de gérer leur panier et de passer commande avec paiement Stripe intégré. Système de rôles (USER, EDITOR, ADMIN) pour la gestion des produits, commandes et utilisateurs. Génération automatique de PDF pour les bons de commande et envoi d'emails de confirmation.

## 2. Application de Gestion du Personnel

**Technologies :** PHP, MySQL, Bootstrap, Architecture MVC

Application web de gestion du personnel développée en PHP avec architecture MVC (Modèle-Vue-Contrôleur). Système d'authentification sécurisé avec hashage SHA-256 et salage des mots de passe. Interface responsive avec Bootstrap permettant la gestion complète des employés (ajout, modification, suppression). Architecture modulaire séparant la logique métier, la présentation et les données pour une maintenabilité optimale.

---

## 3. Coach - Application Mobile .NET MAUI

**Technologies :** .NET MAUI, C#, XAML, MVVM

Application mobile multiplateforme développée avec .NET MAUI permettant de calculer l'Indice de Masse Grasse (IMG) selon la formule de Deurenberg. Interface intuitive avec champs de saisie optimisés pour mobile (poids, taille, âge, sexe). Calcul automatique avec affichage de résultats personnalisés incluant des indicateurs visuels (smileys) et des messages d'interprétation. Application disponible sur Android, iOS, macOS et Windows avec une seule base de code.

---

## 4. Suivi de l'ISS en Temps Réel

**Technologies :** JavaScript, Leaflet, API Where the ISS at, HTML/CSS

Application web de suivi en temps réel de la Station Spatiale Internationale développée avec JavaScript vanilla et la bibliothèque Leaflet. Intégration de l'API Where the ISS at pour récupérer les données de position, altitude et vitesse. Carte interactive avec marqueur personnalisé se déplaçant automatiquement toutes les secondes. Affichage en temps réel des coordonnées géographiques, altitude en kilomètres et vitesse en km/h avec animations visuelles.

---

## 5. Chatbot Openbiz Dev

**Technologies :** JavaScript vanilla, HTML, CSS

Assistant virtuel interactif développé pour l'agence Openbiz Dev permettant de guider les visiteurs dans leur parcours de qualification. Système de conversation structuré avec sélection de services (Étude, Site, Social), qualification via questions ciblées (objectif, budget, délai) et redirection vers Calendly pour la prise de rendez-vous. Interface moderne avec animations fluides et gestion complète de l'état de conversation. Collecte et résumé des informations utilisateur avant redirection vers l'agence appropriée selon la localisation.
