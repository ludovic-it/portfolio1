Suivi de l'ISS en Temps Réel

Application web de suivi de la Station Spatiale Internationale développée avec JavaScript et Leaflet

Présentation du projet

Dans le cadre de ma formation, j'ai développé une application web permettant de suivre en temps réel la position de la Station Spatiale Internationale (ISS) à l'aide de JavaScript, Leaflet et d'une API de géolocalisation. L'objectif était de concevoir une interface web permettant aux utilisateurs de visualiser la position actuelle de l'ISS sur une carte interactive, ainsi que ses coordonnées précises (latitude, longitude), son altitude et sa vitesse.

L'application utilise l'API Where the ISS at pour récupérer les données en temps réel de la station spatiale, et la bibliothèque Leaflet pour afficher une carte interactive. Les informations sont mises à jour automatiquement toutes les secondes, offrant un suivi en temps réel de la trajectoire de l'ISS.

Fonctionnalités principales

L'application affiche une carte interactive centrée sur la position actuelle de l'ISS, avec un marqueur personnalisé représentant la station spatiale. Les utilisateurs peuvent visualiser en temps réel les coordonnées géographiques (latitude et longitude), l'altitude en kilomètres et la vitesse en kilomètres par heure.

La carte est entièrement interactive : les utilisateurs peuvent zoomer, déplacer la vue et explorer la trajectoire de l'ISS autour de la Terre. Le marqueur se déplace automatiquement toutes les secondes pour refléter la nouvelle position de la station.

Cartographie Interactive

Mise à Jour Temps Réel

Actualisation automatique de la position toutes les secondes grâce à setInterval().

Icône personnalisée de l'ISS affichée sur la carte à la position exacte de la station.

Affichage des coordonnées, altitude et vitesse avec formatage précis et animations visuelles.

Intégration de l'API

L'application utilise l'API Where the ISS at (endpoint : https://api.wheretheiss.at/v1/satellites/25544) pour récupérer les données de l'ISS. Cette API retourne un objet JSON contenant les informations de localisation en temps réel, notamment la latitude, la longitude, l'altitude et la vitesse de la station. 

const api_url = 'https://api.wheretheiss.at/v1/satellites/25544';

async function GetISS() {
    const response = await fetch(api_url);
    const data = await response.json();

    const { latitude, longitude, altitude, velocity } = data;
    
    document.getElementById('lat').textContent = latitude.toFixed(2) + '°';
    document.getElementById('lon').textContent = longitude.toFixed(2) + '°';
    document.getElementById('alt').textContent = altitude.toFixed(2) + ' km';
    document.getElementById('speed').textContent = velocity.toFixed(2) + ' km/h';

    marker.setLatLng([latitude, longitude]);
    map.setView([latitude, longitude], 4);
}

Architecture

Le projet utilise une architecture front-end simple et efficace, basée sur JavaScript vanilla pour la logique applicative, Leaflet pour la cartographie interactive, et Bootstrap pour le design responsive. L'application est constituée de trois fichiers principaux : un fichier HTML pour la structure, un fichier CSS pour le style, et un fichier JavaScript pour la logique métier.

Structure du projet

Le projet est organisé de manière claire avec les fichiers suivants :

  • index.html : Structure HTML de l'application avec intégration Bootstrap et Leaflet
  • index.js : Logique JavaScript pour l'appel API, la gestion de la carte et la mise à jour des données
  • style.css : Styles CSS personnalisés avec animations et effets visuels modernes
  • Assets/iss.png : Icône personnalisée de l'ISS utilisée comme marqueur sur la carte

Initialisation de la carte Leaflet

La carte Leaflet est initialisée avec une vue centrée sur les coordonnées [0, 0] et un niveau de zoom de 2, permettant une vue globale de la Terre. Les tuiles OpenStreetMap sont utilisées comme fond de carte. 

const map = L.map('issMap').setView([0, 0], 2);

L.tileLayer('https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png', {
    attribution: '© OpenStreetMap contributors'
}).addTo(map);

Création du marqueur personnalisé

Un marqueur personnalisé est créé à l'aide de l'icône de l'ISS stockée dans le dossier Assets. Les dimensions et l'ancre du marqueur sont définies pour assurer un positionnement précis sur la carte. 

const issIcon = L.icon({
    iconUrl: 'Assets/iss.png',
    iconSize: [50, 32],
    iconAnchor: [25, 16]
});

const marker = L.marker([0, 0], { icon: issIcon }).addTo(map);

Développement

Fonction GetISS() et mise à jour automatique

La fonction GetISS() est une fonction asynchrone qui effectue un appel fetch à l'API pour récupérer les données de l'ISS. Une fois les données récupérées, elle met à jour les éléments DOM affichant les coordonnées, l'altitude et la vitesse, puis repositionne le marqueur sur la carte et centre la vue sur la nouvelle position.

Pour assurer une mise à jour en temps réel, la fonction setInterval() est utilisée pour appeler GetISS() toutes les secondes (1000 millisecondes). Des animations CSS sont appliquées aux valeurs lors de leur mise à jour pour améliorer l'expérience utilisateur. 

async function GetISS() {
    const response = await fetch(api_url);
    const data = await response.json();

    const { latitude, longitude, altitude, velocity } = data;

    const latEl = document.getElementById('lat');
    const lonEl = document.getElementById('lon');
    const altEl = document.getElementById('alt');
    const speedEl = document.getElementById('speed');

    latEl.textContent = latitude.toFixed(2) + '°';
    lonEl.textContent = longitude.toFixed(2) + '°';
    altEl.textContent = altitude.toFixed(2) + ' km';
    speedEl.textContent = velocity.toFixed(2) + ' km/h';

    latEl.classList.add('updated');
    lonEl.classList.add('updated');
    altEl.classList.add('updated');
    speedEl.classList.add('updated');

    setTimeout(() => {
        latEl.classList.remove('updated');
        lonEl.classList.remove('updated');
        altEl.classList.remove('updated');
        speedEl.classList.remove('updated');
    }, 500);

    marker.setLatLng([latitude, longitude]);
    map.setView([latitude, longitude], 4);
}

GetISS();
setInterval(GetISS, 1000);

Design responsive avec Bootstrap

Bootstrap 5.3 est utilisé pour créer une interface responsive et moderne. La grille Bootstrap permet d'organiser les éléments de manière adaptative : les informations sont affichées dans une carte à gauche sur les grands écrans, et empilées verticalement sur les petits écrans. La hauteur de la carte utilise l'unité relative vh (viewport height) pour s'adapter à toutes les tailles d'écran.

Animations et effets visuels

Le CSS personnalisé inclut de nombreuses animations et effets visuels pour améliorer l'expérience utilisateur :

  • Fond animé avec étoiles clignotantes en arrière-plan
  • Animations d'apparition pour les cartes (fadeIn, slideIn)
  • Effets de survol (hover) avec transformations et changements de couleurs
  • Animations sur les valeurs lors de leur mise à jour
  • Dégradés de couleurs pour les en-têtes et les valeurs
  • Point pulsant dans la navbar indiquant le statut en direct
#issMap {
    height: 60vh;
    width: 100%;
    border-radius: 0 0 15px 15px;
}

.value.updated {
    animation: valueUpdate 0.5s ease;
}

@keyframes valueUpdate {
    0% { transform: scale(1); }
    50% { transform: scale(1.15); color: #00ff88; }
    100% { transform: scale(1); }
}
